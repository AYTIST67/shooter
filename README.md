# shooter 
# Космический шутер 🚀

Космический шутер на Python с использованием библиотеки Pygame, где вы управляете ракетой и защищаете себя от атак НЛО и астероидов.

## 📋 Описание

Вы пилот космической ракеты, который должен уничтожить 10 вражеских НЛО, не пропустив больше 3 врагов и не потеряв все жизни. Враги падают сверху экрана, а вы должны стрелять по ним, уклоняясь от столкновений.

## 🎮 Управление

| Кнопка | Действие |
|--------|----------|
| **← Влево** | Движение ракеты влево |
| **→ Вправо** | Движение ракеты вправо |
| **Пробел** | Стрельба пулей |

## 🎯 Цель игры

Набрать **10 очков** (уничтожить 10 НЛО) до того, как:
- Пропустите **3 врага** (они пролетят мимо экрана)
- Потеряете **все 3 жизни** (столкновение с врагом)

## 📊 Показатели

- **Счёт**: количество уничтоженных НЛО (цель — 10)
- **Пропущено**: врагов, пролетевших мимо (лимит — 3)
- **Жизни**: 3 жизни (цвет меняется: зелёный → жёлтый → красный)

## ⚙️ Механики игры

### Стрельба
- Максимум **5 выстрелов подряд**
- После 5 выстрелов — **перезарядка 3 секунды**
- Во время перезарядки появляется надпись **"Wait reload..."**
- Звук выстрела: `fire.ogg`

### Враги

| Тип | Количество | Описание |
|-----|------------|----------|
| **НЛО** | 5 (восстанавливаются) | Дают 1 очко при уничтожении, скорость 1-5 пикселей/кадр |
| **Астероиды** | 2 | Не дают очков, просто препятствие |

### Условие победы

```python
if score >= 10:
    → YOU WIN! (победа)
```

### Условие поражения

```python
if life == 0 or lost >= 3:
    → YOU LOSE! (проигрыш)
```

После конца игры (победа или поражение) автоматический перезапуск через **3 секунды**.
